## Weather

Классическое погодное приложение на основе weatherapi.com.

## Важно: для получения погоды необходимо вставить apikey в файл Constants.

## Стек технологий
- SwiftUI
- iOS 15+
- MVVM
- Combine
- Async/await
- Localization
